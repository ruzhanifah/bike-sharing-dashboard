# 🚲 Bike Sharing Data Analysis Dashboard

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis pola penyewaan sepeda menggunakan dataset Bike Sharing. Analisis difokuskan pada pengaruh cuaca, musim, serta pola waktu terhadap jumlah penyewaan sepeda.

## 📊 Dataset
- **day.csv** → dataset utama (analisis harian)
- **hour.csv** → dataset tambahan (analisis per jam)

## ❓ Pertanyaan Bisnis
1. Bagaimana pengaruh kondisi cuaca dan musim terhadap jumlah penyewaan sepeda?
2. Kapan waktu dengan permintaan tertinggi untuk pengguna registered dibanding casual?

## 🛠️ Tools
- Python
- Pandas
- Seaborn
- Matplotlib
- Streamlit
